# Logistics Order & Delivery Analytics

## Project Description
A MySQL-based DBMS project for managing logistics operations, including customers, orders, shipments, drivers, vehicles, routes, invoices, and payments. 
It demonstrates relational database design, normalization, SQL queries, stored procedures, indexing, and query optimization using MySQL Workbench.

## Features
- Customer Management
- Order Management
- Shipment Tracking
- Driver & Vehicle Management
- Payment Tracking
- SQL Reports
- Stored Procedures
- Query Optimization

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Database Tables
- Customers
- Orders
- Shipments
- Drivers
- Vehicles
- Routes
- Invoices
- Payments
